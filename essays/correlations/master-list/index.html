Correlations: 1600 Correlated Words

What is this list?
If you’ve just arrived to this site from a search engine, this is probably not the best place to start reading. If you’re interested in this approach of philosophical inquiry, you can read the Introduction to Correlations. Otherwise, check out the home page.

When you’re familiar with this method of inquiry, continue on with the following list which correlates around 1600 basic words. The Active (Yang) words are in alphabetical order, with their antonyms in the Passive (Yin) columns. See Couplets for another way to view these words.

First scan the correlations vertically to sense the qualities they share. Then read some horizontally, i.e., “activity abandons“, “passivity restrains“. Next, read some circularly, i.e., “activity abandons passivity,… passivity restrains activity“. Finally, jump around and string words together, i.e., “active life grows & abandons passive death,… passive death restrains & stagnates active life“. These procedures often produce obscure results; please ponder them — it takes time for deeper meaning to emerge. Also, work with those words which intrigue you. If they’re not on this list, add them.
